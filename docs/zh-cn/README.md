## 前言

首先十分感谢您选择并使用本插件。

Dominion 是一个完全开源、免费，专为高版本开发，面向未来的全新领地保护插件。插件目前处于快速的开发、迭代阶段，我们会尽力保证插件的稳定性和功能性。

如果您在使用过程中遇到任何问题，或者有任何建议，欢迎在 [GitHub ISSUE](https://github.com/ColdeZhang/Dominion/issues) 提交您的问题或想法。
