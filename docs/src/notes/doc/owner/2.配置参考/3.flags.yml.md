## 简介

有些特殊的领地权限对很多服务器来说是不需要的，比如创造飞行（fly）对于绝大多数生存服务器来说就是不需要的。
通过 `flags.yml` 文件，可以禁用不需要权限，以更适合你的服务器主题。

- `enable`: 是否启用该权限
- `default`: 该权限的默认值
