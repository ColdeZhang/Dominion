其中 GitHub 平台同时提供了 `lite` 和 `full` 两种版本，其他平台只提供 `lite` 版本。
- `lite` 版本不包第三方依赖库，会在第一次安装后启动服务器时自动下载。
- `full` 版本包含了所有依赖库，可以直接使用。

## 安装

将下载好的插件文件放入服务器的 `plugins` 文件夹中，然后重启服务器即可。
